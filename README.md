# Calculadora de Combustivel ⛽

<p>Projeto em Python guiado pela plataforma DevMedia.</p>

<p>O programa permite calcular a quantidade de combustivel gasto por um veiculo em um determinado percurso de acordo com dados fornecidos pelo usuario.</p><br/>

## Pré-requisitos e execução<br/>

Para executar o codigo será necessário a instalação do [Python 3](https://www.python.org/)

```
# Clone este repositório
$ git clone <https://github.com/>

# Acesse a pasta do projeto no terminal/cmd
$ cd /
$ cd calculo_combustivel

# Execute o arquivo principal
$ python main.py

```
